# Medical Insurance Cost Prediction

## Project Overview
This project predicts medical insurance charges using machine learning models based on features like age, BMI, smoking status, and region.

## Dataset Features
- Age
- Sex
- BMI
- Children
- Smoker
- Region
- Charges (target)

## Models Used
- Linear Regression
- Random Forest Regressor

## Key Insights
- Smoking has the strongest impact on insurance charges
- Age and BMI also affect charges
- Region and sex have relatively lower impact

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
